## 📊 Project Overview

This project provides a **comprehensive stock market analysis pipeline** for 50 Nifty 50 constituent stocks traded on NSE/BSE from **January 2025 to April 2025**. It combines data engineering, technical analysis, database management, and business intelligence to deliver actionable trading insights.

### Business Problem

Retail investors and portfolio managers struggle to:
- **Monitor** multiple stocks across different sectors simultaneously
- **Identify** trading opportunities using technical indicators (RSI, MACD, Bollinger Bands)
- **Track** market trends and sector rotation in real-time
- **Automate** daily data collection and reporting workflows

### Solution

An integrated analytics platform that:
- Fetches and processes 500K+ records of historical stock data
- Computes 15+ technical indicators per stock per day
- Stores data in PostgreSQL for fast querying
- Visualizes insights via Power BI and Tableau dashboards
- Automates daily ETL with Google Apps Script and email alerts

### Key Questions Answered

1. Which sectors show bullish/bearish momentum?
2. What are the top gaining and losing stocks?
3. When do RSI and MACD signal trading opportunities?
4. How volatile is each sector compared to others?
5. Which stocks exhibit RSI divergence (early reversal signals)?

## 🚀 Setup Instructions

### Prerequisites

- **Python 3.10+**
- **PostgreSQL 14+** (or use SQLite for testing)
- **Power BI Desktop** (Windows only)
- **Tableau Desktop** (or Tableau Public)
- **Google Account** (for Apps Script)

### Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/stock-market-analysis.git
cd stock-market-analysis
```

### Step 2: Install Python Dependencies

```bash
cd python
pip install -r requirements.txt
```

### Step 3: Configure Environment Variables

Create a `.env` file in the `python/` directory:

```bash
cp .env.example .env
```

Edit `.env` with your PostgreSQL credentials:

```env
POSTGRES_HOST=localhost
POSTGRES_DB=stock_analysis
POSTGRES_USER=postgres
POSTGRES_PASSWORD=your_password_here
POSTGRES_PORT=5432
LOG_LEVEL=INFO
```

### Step 4: Create PostgreSQL Database & Schema

```bash
# Create database
createdb stock_analysis

# Run schema script
psql -U postgres -d stock_analysis -f sql/schema.sql
```

### Step 5: Run Data Pipeline

```bash
cd python
python data_pipeline.py
```

**Expected Output:**
- Fetches ~500K records from yfinance (50 stocks × ~90 trading days)
- Computes technical indicators (RSI, MACD, Bollinger Bands, etc.)
- Saves to `data/nse_bse_features.csv`
- Takes 10-15 minutes depending on network speed

### Step 6: Load Data into PostgreSQL

```bash
cd sql
python load_data.py
```

**Features:**
- Batch upserts (1000 rows/batch)
- Handles duplicate records
- Verifies load with row counts

### Step 7: Deploy Flask API

```bash
cd api
python flask_api.py
```

API runs at: `http://localhost:5000`

**Endpoints:**
- `GET /api/health` - Health check
- `GET /api/latest` - Latest data for all stocks
- `GET /api/signals` - Trading signals (BUY/SELL)
- `GET /api/sector-summary` - Sector-level aggregates
- `GET /api/stock/<symbol>` - Historical data for one stock

### Step 8: Build Dashboards

**Power BI:**
1. Open Power BI Desktop
2. Follow instructions in [`powerbi/README.md`](powerbi/README.md)
3. Connect to PostgreSQL or import CSV
4. Create DAX measures and visuals

**Tableau:**
1. Open Tableau Desktop
2. Follow instructions in [`tableau/README.md`](tableau/README.md)
3. Build sector drill-down dashboard
4. Publish to Tableau Public (optional)

### Step 9: Set Up Google Apps Script

1. Create new Google Sheet
2. Go to **Extensions** → **Apps Script**
3. Copy contents of:
   - `apps_script/etl_automation.js`
   - `apps_script/menu.js`
4. Update `CONFIG` section with your Flask API URL and Sheet ID
5. Set up daily trigger: **Triggers** → **Add Trigger** → `runDailyETL` → Day timer → 8am-9am
6. Authorize permissions (sheets, email, external API)

## 🔍 Key Findings

Based on analysis of Jan-Apr 2025 data:

### 1. IT Sector Shows Strongest Momentum
- Average RSI: 58.3 (highest among all sectors)
- YTD Return: +12.5% (INFY +15.2%, TCS +12.8%)
- 42% BUY signals in Q1 2025

### 2. Banking Sector RSI Divergence Detected
- In March 2025, HDFCBANK and ICICIBANK made new price highs
- RSI failed to confirm (bearish divergence)
- Sector corrected -3.2% in April

### 3. FMCG Stocks Lowest Volatility
- Average volatility: 18.4% (vs market avg 28.7%)
- Defensive characteristics during market uncertainty
- HINDUNILVR and NESTLEIND most stable

### 4. Energy Sector Highest BUY Signal Frequency
- 48% of observations triggered BUY signals
- Driven by ONGC and COALINDIA oversold conditions (RSI < 40)
- MACD crossovers preceded price moves by 2-3 days on average

### 5. Most Volatile Stocks (Top 5)
1. ADANIENT (45.2% annualized vol)
2. TATASTEEL (41.8%)
3. JSWSTEEL (39.5%)
4. DLF (38.1%)
5. RELIANCE (36.7%)

### 6. Sector Correlation Insights
- IT and Banking: 0.72 (highly correlated)
- FMCG and Energy: 0.23 (low correlation, good for diversification)
- Metals and Realty: 0.68 (move together)

## 📊 SQL Analytical Queries

The project includes 8 advanced SQL queries in [`sql/queries.sql`](sql/queries.sql):

1. **Top 10 stocks by avg daily return per sector** - RANK() window function
2. **BUY/SELL/HOLD count per sector per month** - DATE_TRUNC aggregation
3. **Sector-wise avg RSI and MACD histogram** - Momentum analysis
4. **Most volatile stocks (top 10)** - Risk ranking
5. **MACD bullish crossovers in last 5 days** - LAG() window function
6. **Nifty 50 index-level aggregated metrics** - Market-wide statistics
7. **RSI divergence detection** - Price vs indicator divergence
8. **Monthly performance summary per sector** - Comprehensive aggregation

**Bonus Queries:**
- Sector correlation matrix (returns)
- Best and worst trading days by sector

## 🤖 Google Apps Script Features

### Automated Daily ETL (8:00 AM IST)

1. **Fetches latest data** from Flask API with retry logic (3 attempts)
2. **Updates 4 Google Sheet tabs:**
   - `Raw_Data`: OHLCV + all technical indicators
   - `Signals`: BUY/SELL signals only (conditional formatting)
   - `Sector_Summary`: Aggregated sector metrics
   - `Index_View`: Nifty 50 daily aggregates
3. **Sends email summary** with:
   - Today's BUY/SELL signal counts
   - Top 3 gainers and losers
   - ETL runtime and records processed
4. **Logs each run** with timestamp and status

## 🔧 Troubleshooting

### Common Issues

**yfinance returns no data:**
- Check internet connection
- Verify stock symbols are correct (NSE format: `INFY.NS`)
- Try fetching fewer stocks at a time

**PostgreSQL connection failed:**
- Verify PostgreSQL is running: `pg_isready`
- Check credentials in `.env` file
- Ensure database exists: `psql -l | grep stock_analysis`

**Flask API not starting:**
- Check if port 5000 is available: `netstat -ano | findstr :5000`
- Verify database connection in `.env`
- Check Flask logs for errors

**Google Apps Script fails:**
- Verify Flask API is publicly accessible (not localhost)
- Check API URL in `CONFIG` section
- Review execution logs in Apps Script editor
- Ensure all permissions are granted

**Power BI cannot connect:**
- Install PostgreSQL ODBC driver
- Verify connection string
- Try importing CSV as fallback
